# **Technical specifications:**

| Specification                               | Value                                           |
| ------------------------------------------- | ----------------------------------------------- |
| Roll former is used to produce beams        | Beams are all produced in identical dimensions  |
| Rollers are fixed on a robust base          | To ensure stability during operation            |
| Each roller                                 | is mounted on two roller bearings               |
| Roller sets could be changed                | To work on different cross section or thickness |
| Roll former is operated from electric motor | 12 hp and 380 V                                 |
| Former can form sheet metal                 | Up to 2 mm thickness and width up to 25 cm      |
| Vertical adjustment of rollers              | 50 mm from 80 mm to 130 mm                      |
| Output speed of products                    | 18 m/min                                        |
| Number of stations for rolling              | 12                                              |
| Overall dimensions are                      | 290 cm. Length by 65 cm. width by 104 cm high   |
| Gross weight                                | 750 kg                                          |
